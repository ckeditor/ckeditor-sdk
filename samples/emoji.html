Emoji plugin

Emoji plugin provides possibility to easily insert utf symbols, commonly known as emoji, inside editor content. Every emoji has it's own name which is used to select proper symbol from available set. By default pressing : activates drop down suggestion box where user can select proper emoji. User can type emoji name after : character to narrow down suggestions on the list.

Plugin provides configuration options.

emoji_numberOfCharactersToStart customises how many characters is required to activate emoji suggestion box. For example setting this value to 2 will make emoji's suggestions appear after typing :fa.

Another configuration option is emoji_preventTransformation. This array contains name of tags where emoji is not active. By default emoji plugin is not active inside <pre> and <code> tags

Emoji are integrated with embedding inserted content. That's why strings like :face_with_tears_of_joy: will be automatically convert to emoji symbol during paste event, inserting text, creating editor or setting up editor's data.
