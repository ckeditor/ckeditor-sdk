Balloon Toolbar

The optional Balloon Toolbar plugin, introduced in CKEditor 4.8, allows for adding context-aware toolbar in the editor.

This feature provides you with a very easy way of defining customized toolbar for a given content type.

As an example, below you'll find an editor where following buttons are displayed:

  • Link, Unlink and Image buttons for image widgets.
  • Bold button for placeholder widgets.
  • Numbered List and Bulleted List buttons for list items.
  • Bold, Italics and Strike buttons for any other case.

Please, give it a try - put the selection in a list, image or any other place of the editor content to see a different toolbar.

Please note that this plugin is not compatible with Internet Explorer 8.
