Creating Easy Images Documentation

The optional Easy Image plugin, introduced in CKEditor 4.9, supports uploading and inserting images into editor content. It provides the following features when compared to the other Image widget that is available as optional plugin in CKEditor distributions:

  • It allows for adding image captions (that will not be separated from the image when its location changes).
  • It allows for uploading images to Cloud Services, which optimizes images by rescaling them and compressing, resulting in smaller sizes and faster download times.
  • It produces responsive markup, so inserted images are suitable for a wide range of devices (including mobile ones).
  • It allows for adding custom styles for inserted images.
  • It uses customizable balloon toolbar for every widget.
  • It does not produce any reflow, resulting in smoother working, visible to the user.

A few configuration options are available to fine-tune this feature, including adding custom styles, customizing widget's toolbar or setting a custom CSS class for the widget. Refer to the Creating Easy Images article to learn more about this feature.
